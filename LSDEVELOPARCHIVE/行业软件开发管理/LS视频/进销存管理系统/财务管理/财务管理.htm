财务管理

财务管理包括：

（一）基础数据维护：银行账户信息管理、业务凭证分录信息的设置及业务凭证分录信息的修改

n         银行账户信息：银行账户信息的新增，编辑，作废

（二）初始化：对会计期间信息的设置、会计科目信息的设置、科目余额初始化及结束财务信息设置初始化

n         会计期间设置：会计期间的添加，删除

n         会计科目设置：设置本企业所需要的会计科目

n         科目余额初始化：录入企业会计科目余额

n         结束初始化：以上信息处理完后，结束操作

（三）凭证管理：手工凭证、业务凭证 登记；凭证审核、手工凭证的修改、凭证过账及其查询

n         登记手工凭证：手工凭证登记

n         登记业务凭证：业务凭证登记

n         审核凭证：凭证审核（审核，全审，反审）

n         修改手工凭证：手工凭证的编辑

n         凭证过账：凭证过账（过账，全过账，反过账）

n         凭证查询：凭证信息查询

（四）资金管理：对收款、付款、预收账款、预付账款的 业务处理及票据管理（包括：票据查询、转让及兑付）

n         收款：依据销售结算或采购退货结算进行收款

n         付款：依据采购结算或销售退货结算进行付款

n         预收账款：预先收款操作

n         预付账款：预先付款操作

n         票据管理：对企业收到或者付出的票据的记录过程

      1.票据查询：查询企业收到或付出的票据

      2.票据转让：收到的票据再转让给别的企业的登记

      3.票据兑付：收到或付出的票据到期或贴现兑付操作

（五）期末结账：科目汇总表、结转损益、期末结账

n         科目汇总表：所有科目借方发生额与贷方发生额的汇总

n         结转损益：结转本企业的损益类科目

n         期末结账：结束一个会计期间的业务同时把所有帐簿的余额结转到下一期

（六）发票管理：发票登记

n         发票登记：登记发票

（七）账薄查询：对总账明细账、现金日记账、 银行日记账信息的查询、分类总账明细账查询

n         总账明细账查询：查询企业的所有总账，明细账

n         现金日记账查询：企业的现金日记账

n         银行日记账查询：企业的银行日记账

n         分类总账明细账查询：查询企业的某一总账，明细账
